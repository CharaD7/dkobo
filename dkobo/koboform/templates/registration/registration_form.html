KoBoToolbox is an integrated set of tools for building forms and collecting interview responses. It is built by the Harvard Humanitarian Initiative for easy and reliable use in difficult field settings, such as humanitarian emergencies or post-conflict environments.

This installation of KoBoToolbox may only be used for small survey deployments, which means less than 1,500 submissions per user per month. If you require more, please contact us.

If you are a organization providing humanitarian assistance, please use OCHA's KoBoToolbox installation instead, which provides an unlimited number of submissions.

You can also download and install KoBoToolbox on your own server or on your own computer.
